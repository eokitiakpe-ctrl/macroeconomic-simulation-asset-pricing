# Macroeconomic-simulation-asset-pricing
Macroeconomic simulation using a Markov model to analyze asset pricing and policy impacts

This project builds a Markov-based macroeconomic model using FRED data to analyze how economic conditions evolve over time and impact asset pricing and economic outcomes.

## Key Features
- Markov state-transition modeling
- Asset pricing using stochastic discount factor
- Simulation of COVID-19 policy scenarios
- Analysis of economic welfare and risk

## Tools
Python (NumPy, Pandas, Matplotlib)

## Key Insight
Economic shocks and policy interventions significantly affect long-run consumption, asset prices, and economic stability.
